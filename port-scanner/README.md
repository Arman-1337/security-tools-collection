# Port Scanner

A multi-threaded Python port scanner for network security testing.

## Features

- Multi-threaded scanning for fast performance
- Customizable port ranges
- Service detection for common ports
- Clean, formatted output
- IPv4 and hostname support

## Usage
```bash
python port_scanner.py
```

Then follow the prompts to enter:
- Target IP/hostname
- Start port (default: 1)
- End port (default: 1024)

### Example
```bash
python port_scanner.py

Enter target IP/hostname: scanme.nmap.org
Enter start port: 1
Enter end port: 1000
```

## Requirements

- Python 3.6+
- No external dependencies (uses only standard library)
